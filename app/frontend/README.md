# DocLint Frontend

Vue 3 + TypeScript frontend for the DocLint PDF document format validator.

## Tech Stack

- **Vue 3** + **TypeScript** + **Vite**
- **Vue Router**
- **Pinia**
- **PrimeVue** + **Tailwind CSS**
- **Axios**

## Prerequisites

- Node.js 20+

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Opens at `http://localhost:5173`.

## Build

```bash
npm run build
```

Produces the production bundle in `dist/`.
